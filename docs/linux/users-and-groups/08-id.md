# id Command Documentation

## Name and Synopsis

Print real and effective user and group IDs for the current process or a specified user.

**Basic Usage:**

```bash
id [OPTION]... [USER]...
```

---

## Description

The `id` command is used to display identification information for a specific user or the current process. It prints the UID (User ID), GID (Group ID), and group memberships. If no user is specified, the command defaults to providing information for the current process. The utility provides a useful set of identified information by default, including the real and effective IDs.

---

## Options Reference

| Option | Long Form | Description |
| :--- | :--- | :--- |
| `-a` | | Ignore, for compatibility with other versions |
| `-Z` | `--context` | Print only the security context of the process |
| `-g` | `--group` | Print only the effective group ID |
| `-G` | `--groups` | Print all group IDs |
| `-n` | `--name` | Print a name instead of a number for -u, -g, and -G |
| `-r` | `--real` | Print the real ID instead of the effective ID (with -u, -g, -G) |
| `-u` | `--user` | Print only the effective user ID |
| `-z` | `--zero` | Delimit entries with NUL characters (not permitted in default format) |
| `--help` | | Display help information and exit |
| `--version` | | Output version information and exit |

---

## Configuration / Files Modified

This command is a query utility; it does not modify any configuration variables or system files.

---

## Examples

* **Display current user ID, group ID, and groups:**

```bash
id
```

* **Display only the effective user ID:**

```bash
id -u
```

* **Display all group IDs for a specific user:**

```bash
id -G username
```

* **Display the name of the current primary group:**

```bash
id -gn
```

---

## Exit Values

* **0**: Success.
* **Non-zero**: Indicates an error occurred during execution (General behavior).

---

## Caveats / Notes

* **Delimiter Restriction:** The `-z` or `--zero` option, which delimits entries with NUL characters instead of whitespace, is not permitted when using the default format.
